## 2. What is DuckDB?
DuckDB is an **embedded analytical database** designed for OLAP workloads.

Key characteristics:
- No server required (runs in-process)
- Optimized for analytical queries
- Can query **CSV / Parquet files directly**
- Uses standard SQL

DuckDB is often used for:
- Local analytics
- Lightweight ETL pipelines
- Data exploration on files
