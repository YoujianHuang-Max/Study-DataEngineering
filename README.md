# Overview
This repository records daily learning progress in Data Engineering. It includes both conceptual understanding and hands-on practice, covering topics such as:

ETL / ELT processes

Data pipelines and workflow orchestration

Batch and streaming data processing

Data warehouses and lakehouses

Python and PostgreSQL projects

The goal is to build a strong foundation in data engineering principles and practical skills, with structured notes and mini-projects for each day.

# Daily Learning Structure
Each day contains:

1. Concept Notes – Key ideas and definitions in a concise format.

2. Hands-On Practice – Python/PostgreSQL exercises or mini projects.

3. References – YouTube videos, articles, or tutorials used for study.
